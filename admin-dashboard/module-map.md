# Admin Dashboard v2 Module Map

| Module | Job | Primary action |
|---|---|---|
| Review Queue | Show partner/lead items requiring review. | Assign, pause, update status |
| Partner Profiles | Show partner health, tier, risk, next action. | Open profile |
| Lead Queue | Show received-for-review lead submissions. | Review or request info |
| Tracking Analytics | Show safe partner/campaign attribution. | View campaign |
| Sync Monitor | Show Notion/HubSpot sync health. | Retry failed sync |
| Compliance Flags | Show risky copy/output/notes. | Rewrite or escalate |
| GPT Action Logs | Show GPT Action usage. | Inspect failure |
| Tally Intake Monitor | Show webhook intake status. | Replay/review |
| Operator Notes | Show internal notes metadata. | Add note |
| Launch QA Status | Show launch checklist progress. | Resolve blocker |

No module should calculate payouts, expose commission data, or claim partner performance certainty.
