# Campaign Kit Library

## Broker → Lead Submission Kit

- CTA: Submit a client for review.
- Best channel: CRM/email.
- Core asset: lead-submission checklist.
- Guardrail: “Received for review” only. No lender review certainty.

## CPA/bookkeeper → Client Readiness Kit

- CTA: Check funding readiness.
- Best channel: newsletter/client meeting.
- Core asset: cash-flow gap checklist.
- Guardrail: educational framing only.

## Affiliate → Compliant Promo Kit

- CTA: Request funding options review.
- Best channel: short-form content, newsletter, landing page.
- Core asset: compliant promo copy and tracking link guide.
- Guardrail: no income or commission guarantees.

## Attorney → Intro/Referral Kit

- CTA: Request partner access.
- Best channel: direct email intro.
- Core asset: readiness handoff template.
- Guardrail: no legal/financial advice claims.

## Veteran connector → Community Education Kit

- CTA: Apply to become a partner.
- Best channel: workshop, community list, local event.
- Core asset: veteran-owned business funding readiness outline.
- Guardrail: no special guaranteed outcomes.

## Strategic partner → Co-Marketing Kit

- CTA: Request strategic partner review.
- Best channel: webinar, integration page, partner marketplace.
- Core asset: co-marketing workflow map.
- Guardrail: no public API promises until developer access is approved.
