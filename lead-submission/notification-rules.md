# Lead Submission Notification Rules

Notifications are not sent in Batch 36. This file defines when Batch 38 may send or draft alerts.

## Internal alert triggers

- new lead submitted
- duplicate detected
- consent missing
- sensitive data flagged
- manual admin decision needed
- sync failure after lead storage

## Partner-facing messages

Partner-facing messages should be optional, guarded, and reviewed.

Allowed:

- submission received for review
- missing information request
- resource recommendation

Not allowed:

- lender reviewed this lead
- lead is approved
- guaranteed funding
- guaranteed commission
- guaranteed timeline
