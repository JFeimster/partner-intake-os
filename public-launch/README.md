# Public Partner Launch Layer

Batch 45 creates the public-facing partner acquisition layer for Partner Intake OS.

## Purpose

Convert partner interest into structured intake without exposing internal admin systems.

## Pages

- `/site/partner-intake/public-signup.html`
- `/site/partner-intake/partner-types.html`
- `/site/partner-intake/resources.html`
- `/site/partner-intake/faq.html`
- `/site/partner-intake/developer-access.html`

## Approved CTAs

- “Apply to become a partner”
- “Request partner access”

## Forbidden CTAs

- “Get approved instantly”
- “Start earning guaranteed commissions”
- “Everyone qualifies”

## Guardrail

Public pages should educate and route. They should not expose admin routes, API tokens, internal partner records, borrower data, or payout/commission logic.
