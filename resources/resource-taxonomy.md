# Resource Taxonomy

## Resource types

- `lead_submission_kit`
- `client_readiness_kit`
- `compliant_promo_kit`
- `intro_referral_kit`
- `community_education_kit`
- `co_marketing_kit`
- `tracking_link_guide`
- `funding_product_summary`
- `workflow_playbook`
- `disclosure_language`

## Personalization dimensions

- partner type
- tier
- risk level
- onboarding path
- primary audience
- campaign channel

## Resource examples

| Partner type | Resource |
|---|---|
| Broker | Lead submission kit |
| CPA/bookkeeper | Client readiness kit |
| Affiliate | Compliant promo kit |
| Attorney | Intro/referral kit |
| Veteran connector | Community education kit |
| Strategic partner | Co-marketing kit |

## Copy guardrails

Resources should use readiness-based and review-based language. Never imply guaranteed approvals, guaranteed commissions, guaranteed funding amounts, guaranteed timelines, lender certainty, or universal qualification.
