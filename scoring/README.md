# Scoring + Tier Calibration

Batch 41 makes partner scoring tunable, explainable, and reviewable.

## Goals

- Produce a clear score.
- Show “why this score.”
- Let operators override scores without pretending performance is guaranteed.
- Keep risk separate from upside.
- Preserve tier history.

## Outputs

- overall score
- recommended tier
- risk level
- score breakdown
- why-this-score narrative
- manual review triggers
- override metadata

## Guardrail

Scoring is an internal prioritization tool. It is not a prediction of revenue, partner performance, approval, commissions, funding, or outcomes.
