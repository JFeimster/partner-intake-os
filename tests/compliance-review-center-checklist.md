# Compliance Review Center Checklist

## Scanner checks

- [ ] Flags “guaranteed approval”.
- [ ] Flags “everyone qualifies”.
- [ ] Flags “instant funding”.
- [ ] Flags “risk-free”.
- [ ] Flags “no documents needed”.
- [ ] Flags “guaranteed commission”.
- [ ] Flags “earn guaranteed income”.

## Rewrite checks

- [ ] Provides safer rewrite for each restricted phrase.
- [ ] Adds disclosure when missing.
- [ ] Marks high/blocked issues as manual review required.

## Queue checks

- [ ] Admin compliance queue requires auth.
- [ ] Queue item has severity.
- [ ] Queue item has source type and source ID.
- [ ] Decisions are audit-log-ready.
