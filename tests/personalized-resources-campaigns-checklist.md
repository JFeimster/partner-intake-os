# Personalized Resources + Campaigns Checklist

## Personalization checks

- [ ] Broker input returns lead submission kit.
- [ ] CPA/bookkeeper input returns client readiness kit.
- [ ] Affiliate input returns compliant promo kit.
- [ ] Attorney input returns intro/referral kit.
- [ ] Veteran connector input returns community education kit.
- [ ] Strategic partner input returns co-marketing kit.
- [ ] High-risk partner receives compliance review pack instead of promo kit.

## Copy safety checks

- [ ] No deceptive urgency.
- [ ] No guaranteed approvals.
- [ ] No guaranteed commissions.
- [ ] No “everyone qualifies.”
- [ ] Disclosure is included.
